# 🚀 播客解答器 - 一键部署指南

## 方案1：使用现有GitHub账号（推荐）

既然你的 `podcast-knowledge-base` 仓库已经有代码了，现在只需启用GitHub Pages：

### 📋 启用步骤：
1. 访问：https://github.com/zzstart2/podcast-knowledge-base/settings
2. 滚动到 **Pages** 部分
3. Source 选择：**Deploy from a branch**
4. Branch 选择：**main**
5. Folder 选择：**/ (root)**
6. 点击 **Save**

### 🌐 访问地址：
几分钟后即可访问：**https://zzstart2.github.io/podcast-knowledge-base/**

## 方案2：创建新GitHub账号部署

如果你想要一个专门的播客项目账号：

### 🔧 快速注册流程：

#### 1. 创建临时邮箱（无需手机号）
- 访问：https://temp-mail.org/
- 复制获得的临时邮箱地址

#### 2. 注册GitHub账号
- 访问：https://github.com/signup
- 邮箱：使用临时邮箱
- 用户名：`podcast-qa-demo` 或类似
- 密码：设置安全密码

#### 3. 验证邮箱
- 返回 temp-mail.org 查收验证邮件
- 点击验证链接

#### 4. 创建仓库并上传
- 创建新仓库：`podcast-qa-system`
- 设为 Public
- 上传 `index.html` 文件

#### 5. 启用GitHub Pages
- 仓库设置 → Pages → 启用
